# Service_purchase_prediction
This repo is a part of my portfolio of data science projects completed by me for academic, self learning, and hobby purposes.
____
Этот репозиторий создан для решения задачи предсказания вероятности подключения услуги.

* Task.docx - файл с постановкой задачи.
* Service_purchase_prediction.ipynb - ноутбук с решением задачи.
* model.sav - сериализованная модель.
* model_test.ipynb - ноутбук для тестирования сериализованной модели.
* answers_test.csv - файл с предсказанными вероятностями.
* model_description_prezentation.pdf - презентация с описанием.
